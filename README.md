# CCNA-networking-foundations
## Overview
This repository contains networking labs completed while building foundational networking skills at the CCNA level. The focus is on practical congiguration, traffic flow, and troubleshooting.
The purpose is to stregthen core networking knowledge in preparation for cloud security roles.

---

## Topics Covered
- IPv4 addressing and subnetting (CIDR)
- VLAN configuration and trunking
- Inter-VLAN routing
- Static routing
- OSPF (single-area)
- Access Control Lists (ACLs)
- NAT and PAT
- Basic network troubleshooting

---

## Lab Structure
Each lab folder includes:
- Objective
- Topology overview
- Configuration summary
- Verification steps
- Key takeaways

Labs are organized sequentially to reflect progressive complexity.

---

## Tools Used
- Cisco Packet Tracer
- CLI-based router and switch configuration

---

## Goal
To develop strong networking fundamentals that directly support cloud architecture design, VPC segmentation, routing logic, and security boundary implementation.
